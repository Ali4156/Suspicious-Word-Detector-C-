## Project Overview
Suspicious Word Detector is a simple C++ program that analyzes a message or text and identifies potentially suspicious or phishing-related keywords.

This project demonstrates a basic concept used in cybersecurity systems to detect spam or malicious messages.

The program scans the input text and checks for common suspicious words such as "hack", "attack", "password", "free", "win", and "urgent".

If any suspicious word is found, the program alerts the user.

## Features
- Detects suspicious keywords in a message
- Demonstrates a basic phishing detection concept
- Beginner-friendly C++ code
- Simple and lightweight program
- Useful for learning basic text analysis

## Technologies Used
- C++
- Standard Library
- String Handling

## How the Program Works
1. The program asks the user to enter a message or text.
2. It scans the text for suspicious or phishing-related keywords.
3. If a suspicious word is detected, the program displays a warning.
4. If no suspicious words are found, the message is considered safe.
